Software

Data visualization

  1. spiralize: Visualize data on spirals.
  2. circlize: circular visualization in R.
  3. ComplexHeatmap: Flexibly arranging multiple heatmaps and self define annotation graphics.
  4. InteractiveComplexHeatmap: Makes interactive ComplexHeatmap.
  5. EnrichedHeatmap: Visualizes the enrichment of genomic signals on specific target region.
  6. HilbertCurve: Making 2D Hilbert Curve.
  7. gtrellis: genome-level trellis layout.
  8. BioCartaImage: BioCarta pathway images.

Data analysis

  1. sfcurve: 2x2, 3x3 and nxn Space-Filling Curves.
  2. simona: Semantic Similarity in Bio-Ontologies.
  3. simplifyEnrichment: Simplify functional enrichment results.
  4. cola: A Framework for Consensus Partitioning.
  5. rGREAT: Client of GREAT analysis.
  6. CePa: centrality-based pathway enrichment.

Data packages

  1. GeneSummary: RefSeq Gene Summaries.
  2. UniProtKeywords: Keywords from the UniProt database.
  3. BioMartGOGeneSets: Gene Ontology Gene Sets from BioMart.

Teaching

  1. GSEAtraining: Learning gene set enrichment analysis.

Other tools

  1. pkgndep: Check the Heaviness of Package Dependencies.
  2. bsub: Submitter and monitor of LSF cluster.
  3. GetoptLong: wrapper of the Perl module Getopt::Long, also provides a simple variable interpolation in R.
  4. GlobalOptions: setting global options in R.
